Download

If you are still having problems with the license please do not hesitate to contact us.

For some plugins you may need extra packages. You can download the archives from the box on the right side of your browser window. Also have a look at the documentation before asking for help.

Extra Packages

These extra packages may not be available under GPL and may have restrictions. Read the documentation for the packages or look at their home page for further information.
 
Cryptix ()
[Non-GPL] This is a modified version of the original Cryptix cryptography package that is needed for the SSH plugin.
Connection Relay ()
[GPL] This package contains two relay daemons to be installed on the web server. They forward applet connections to other hosts and thus allow you to navigate around the applet security restrictions.

relayd: A program that will forward to one specific host only.
mrelayd: A program that uses the relay settings of the Socket plugin and allows connections to any host, so be careful!

If you have a UNIX box, compile the source files. For NT users we have provided executables in the package. The package can be unpacked with any ZIP compatible program (like winzip) or the Java jar utility.
